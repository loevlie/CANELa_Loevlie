Welcome to the Denny Docs!

CANELa_Loevlie is a collection of Python codes for members of the CANELa lab. The hope is this documentation will help users understand and improve upon Dennis Loevlie’s work from 2021-2022.

This documentation will explain and provide examples for how to perform various tasks using the code developed in the CANELa group. This will involve use of the ce_expansion package developed mainly by James Dean and Michael Cowan and the CANELa_NP and cp2k_helper packages developed by Dennis Loevlie.

Check out the Usage section for further information, including the Installation instructions for the project.

Note

This project is under active development.
